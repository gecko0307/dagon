    Class GLTFAsset

    Asset class for loading and managing GLTF 2.0 resources.

    class GLTFAsset
      : Asset
      , TriangleSet;

    Methods

    NameDescription
    animation (name) Finds an animation by name.
    loadThreadSafePart (filename, istrm, fs, mngr) Loads the thread-safe part of the GLTF asset (parsing JSON, loading buffers, images, etc.).
    loadThreadUnsafePart () Loads the thread-unsafe part (OpenGL resource creation).
    markTransparentEntities () Marks entities as transparent if their material uses transparency.
    material (name) Finds a material by name.
    mesh (name) Finds a mesh by name.
    node (name) Finds a node by name.
    opApply (dg) Iterates over all triangles in the nodes.
    release () Releases all resources associated with the asset.
    scene (name) Finds a scene by name.
    skin (name) Finds a skin by name.

    Description

    Loads buffers, buffer views, accessors, images, textures, materials, meshes, nodes, skins, animations, and scenes from a *.gltf + *.bin pair. *.glb files are not supported.
