    Class TextureAsset

    Represents a texture asset.

    class TextureAsset
      : Asset;

    Constructors

    NameDescription
    this (owner) Constructs a new texture asset.

    Fields

    NameTypeDescription
    buffer TextureBufferTemporary exchange buffer.
    bufferDataIsImageData boolSet to true in the loader if buffer.data refers to image.data (to prevent double-free)
    conversion ConversionOptionsFormat-specific conversion options.
    extension stringFile extension.
    filename stringFilename in the virtual filesystem.
    generateMipmaps boolIf true, generate mip levels.
    image dlib.image.image.SuperImageThe dlib image (optional).
    loadAs3D boolIf true, the input image will be interpreted as 3D texture
    loaderOption uintLoader-specific option.
    persistent boolIf true, keep buffer.data after sending to GPU.
    resolution3D uint
    texture TextureThe GPU texture object.
    monitorInfo MonitorInfoFile monitoring information for the asset.
    threadSafePartLoaded boolIs thread-safe phase of an asset loading is done.
    threadUnsafePartLoaded boolIs thread-unsafe phase of an asset loading is done.

    Methods

    NameDescription
    loadThreadSafePart (filename, istrm, fs, assetManager) Loads the thread-safe part of the texture asset (e.g., from file or stream).
    loadThreadUnsafePart () Loads the thread-unsafe part of the texture asset (e.g., GPU upload).
    release () Releases the GPU texture and all associated resources.
    releaseBuffer () Releases the temporary buffer and image data.

    Description

    The TextureAsset class loads an image to the GPU texture using a TextureLoader defined for the given file extension in the AssetManager.
