    Class Scene

    Manages the application context, assets, world, and environment for a 3D scene.

    Description

    The Scene class provides methods for adding assets (textures, OBJ, GLTF, text, binary), creating entities, cameras, lights, and decals, and handling scene loading and update logic. It supports threaded asset loading, progress reporting, and customizable hooks for loading and update events.
