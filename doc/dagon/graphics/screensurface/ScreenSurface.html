    Class ScreenSurface

    A drawable screen-aligned quad for fullscreen rendering and post-processing.

    class ScreenSurface
      : dlib.core.ownership.Owner
      , Drawable;

    Constructors

    vertices dlib.math.vector.Vector!(float,2)[4]Quad vertex positions in normalized device coordinates.

    Methods

    NameDescription
    render (state) Renders the screen-aligned quad using the provided graphics pipeline state.

    Description

    The ScreenSurface class manages vertex and texture coordinate buffers, OpenGL VAO/VBO/EBO, and provides a render method for drawing the quad. Useful for post-processing, compositing, and screen-space effects.
