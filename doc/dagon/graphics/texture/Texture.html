    Class Texture

    Represents an OpenGL texture object.

    class Texture
      : dlib.core.ownership.Owner;

    NameDescription
    this (owner) Constructs a new texture object.

    channelSize[get] uintReturns the size of a channel in bytes.
    dimension[get] TextureDimensionReturns the texture dimension.
    enableRepeat[set] boolEnables or disables texture repeat on UV wrap.
    hasAlpha[get] boolReturns true if the texture has an alpha channel.
    height[get] uintReturns the height of the texture.
    isCompressed[get] boolReturns true if the texture is compressed.
    isCubemap[get] boolReturns true if the texture is a cubemap.

    Methods

    NameDescription
    bind () Binds the texture to the current OpenGL texture unit.
    createBlank (w, h, channels, bitDepth, genMipmaps, fillColor) Creates a blank texture with the specified parameters and fill color.
    createFallbackTexture () Generate a default texture; unimplemented
    createFromBuffer (buff, genMipmaps) Creates a texture from a buffer.
    createFromEquirectangularMap (envmap, resolution, generateMipmaps) Creates a cubemap texture from an equirectangular environment map.
    createFromImage (img, genMipmaps) Creates a texture from a 2D image.
    createFromImage3D (img, size) Creates a 3D texture from a 2D image
    generateMipmap () Generates mipmaps for the texture.
    release () Releases OpenGL resources.
    setFaceBit (face) Sets the bitmask for a cubemap face.
    setFaceImage (face, img) Sets the image for a cubemap face.
    unbind () Unbinds the texture.
    valid () Returns true if the texture is valid.
    createCubemap (buffer) Creates a cubemap texture from a buffer.
    createTexture1D (buffer) Creates an 1D texture from a buffer.
    createTexture2D (buffer) Creates a 2D texture from a buffer.
    createTexture3D (buffer) Creates a 3D texture from a buffer.

    Description

    Supports 1D, 2D, 3D, and cubemap textures, compressed/uncompressed formats, mipmaps, and filtering. Provides methods for creation from dlib images, buffers, and equirectangular maps, as well as OpenGL resource management.
