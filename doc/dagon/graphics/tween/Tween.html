    Struct Tween

    Tween struct for animating entity properties over time.

    struct Tween;

    Constructors

    NameDescription
    this (entity, type, start, end, duration, easing) Constructs a vector tween for position, rotation, or scaling.
    this (entity, type, start, end, duration, easing) Constructs a color tween.
    this (entity, type, start, end, duration, easing) Constructs a float tween (e.g., opacity).

    Fields

    NameTypeDescription
    active boolIf true, the tween is active.
    dataType TweenDataTypeThe type of data being tweened.
    duration doubleDuration of the tween in seconds.
    easing EasingThe easing function used.
    entity EntityThe entity being animated.
    isPlaying boolIf true, the tween is playing.
    repeat uintNumber of times to repeat the tween.
    repeatCounter uintCurrent repeat count.
    time doubleCurrent time in the tween.
    type TweenTypeThe property being tweened.

    Methods

    NameDescription
    applyTween (t) Applies the tweened value to the entity at the given normalized time.
    ease (t) Calculates the eased time value according to the selected easing function.
    kill () Kills the tween and resets its state.
    pause () Pauses the tween.
    play () Resumes the tween.
    restart () Restarts the tween from the beginning.
    update (dt) Updates the tween by the given time delta.

    Description

    Supports position, rotation, scaling, and color interpolation with various easing functions. Tweens can be paused, played, restarted, repeated, and killed.
