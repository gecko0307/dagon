    Struct DProperty

    Represents a single property value with type information.

    struct DProperty;

    Fields

    NameTypeDescription
    data stringThe property value as a string.
    name stringThe property name.
    type DPropTypeThe property type.

    Methods

    NameDescription
    toBool () Returns the property value as a bool.
    toColor4f () Returns the property value as a Color4f.
    toDouble () Returns the property value as a double.
    toFloat () Returns the property value as a float.
    toInt () Returns the property value as an int.
    toString () Returns the property value as a string.
    toUInt () Returns the property value as an unsigned int.
    toVector3f () Returns the property value as a Vector3f.
    toVector4f () Returns the property value as a Vector4f.

    Description

    Provides conversion methods to various types (string, double, float, int, bool, vector, color).
