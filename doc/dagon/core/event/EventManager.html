    Class EventManager

    Manages event polling, state, and dispatch.

    class EventManager;

    Constructors

    NameDescription
    this (win, winWidth, winHeight) Constructs an EventManager for the given SDL window and dimensions.

    Methods

    NameDescription
    addEvent (e) Adds a new event to the event stack.
    addUserEvent (e) Adds a user event to the user event stack.
    aspectRatio () Returns the current window aspect ratio.
    exit () Signals the event manager to stop running (application exit).
    gameControllerAvailable () Returns true if a game controller is available.
    gameControllerAxis (axis) Gets the normalized value of a game controller axis.
    gameControllerClose (deviceIndex) Closes the game controller or joystick at the specified device index.
    gameControllerOpen (deviceIndex) Opens a game controller or joystick at the specified device index.
    gameControllerRumble (lowFreq, hiFreg, duration) Triggers controller rumble (vibration) if supported.
    generateFileChangeEvent (filename) Generates a file change event for the given filename.
    generateLogEvent (level, message) Generates a log event with the given log level and message.
    generateUserEvent (code) Generates a user event with the given code.
    joystickAvailable () Returns true if a joystick is available.
    joystickAxis (axis) Gets the normalized value of a joystick axis.
    resetUpDown () Resets all UP and DOWN input event states.
    setMouse (x, y) Sets the mouse cursor position.
    setMouseToCenter () Centers the mouse cursor in the window.
    setRelativeMouseMode (mode) Enables or disables relative mouse mode.
    showCursor (mode) Shows or hides the mouse cursor.
    update () Polls and processes all pending events.
    updateTimer () Updates the internal timer and computes delta time.

    Description

    Handles input events, window events, controller/joystick events, and user-defined events. Provides methods for polling SDL events, generating custom events, and querying input state.
