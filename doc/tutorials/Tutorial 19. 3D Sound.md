# Tutorial 19. 3D Sound

Starting from Dagon 0.32 the engine includes a sound playback extension, dagon:audio, which is based on a lightweight yet powerful [SoLoud](https://github.com/jarikomppa/soloud) engine. It supports both 2D and 3D sounds, and the extension makes working with it very easy.
